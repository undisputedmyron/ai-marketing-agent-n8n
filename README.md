# AI Marketing Performance Agent (n8n)

This repository contains an automated AI workflow that reads marketing data from Google Sheets, analyzes performance, and sends email reports using Gemini AI.

## Features
- Marketing performance analysis
- CTR, conversion and ROI calculation
- Automated reporting via Gmail
- Data ingestion from Google Sheets
- AI-powered summary generation
- Memory for context management

## Tools Used
- n8n
- Gemini AI
- Google Sheets API
- Gmail API

## How To Use
1. Import JSON into your n8n instance
2. Configure API credentials:
   - Gemini
   - Google Sheets
   - Gmail
3. Update Sheet ID and column mappings
4. Activate workflow

## Use Cases
- Marketing agencies
- Freelancers
- SaaS dashboards
- Client reporting automation
